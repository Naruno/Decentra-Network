# Prerequisites

Before you begin, make sure you have the following:

## Building Folder Structure

```bash
mkdir test_enviroment
cd test_enviroment
git clone https://github.com/Naruno/Naruno.git
```

## Installing requirements

```bash
pip3 install naruno-tests 
```
